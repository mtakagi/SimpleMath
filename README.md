このプロジェクトは自己学習用に [Swift](https://swift.org) で実装した簡単な数学ライブラリです。
現在 **Vector2**, **Vector3**, **Vector4**, **Quaternion**, **Matrix4x4** が実装されています。

This is a simple math library implemented in Swift for self-study purposes.
It currently includes implementations for **Vector2**, **Vector3**, **Vector4**, **Quaternion**, and **Matrix4x4**.

次のプロジェクトを参考に作成しました。I created this project with reference to the following project:

- [DirectXMath](https://github.com/microsoft/DirectXMath)
- [Unity-Technologies/Unity.Mathematics](https://github.com/Unity-Technologies/Unity.Mathematics)
- [Unity-Technologies/UnityCsReference](https://github.com/Unity-Technologies/UnityCsReference)
